# app-cli

帮助您创建干净的 plugin/doc 模版，支持多种打包工具

- vue
- react
- gitbook
- docsify

构建工具可选

- rollup
- webpack
- vite

## how to use

``` bash
# 全局安装 create-app
$ npm install create-app -g

# 创建项目
$ app-cli init <app-name>
```
