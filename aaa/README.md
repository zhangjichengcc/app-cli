# Docsify

> 开箱即用的文档框架

## 说明

本文档框架基于 docsify，添加了一些常用的 plugin 及一些样式的兼容调整，具体使用可参考官网[docsify 官网](https://docsify.js.org/#/)

## 使用

``` shell
# 安装全局依赖
npm i docsify-cli -g

# 启动本地server
npm start
```
